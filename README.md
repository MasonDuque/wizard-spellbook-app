# wizard-spellbook-app

Immersive, page-by-page D&D wizard spellbook app.

## Highlights
- Leather-textured spellbook cover with wizard name, level, and subclass.
- Dedicated **Open the Book** action from the cover to Page 1.
- Real page-turn behavior: only one page is visible at a time.
- Table of contents, prepared spells page, learn-spell page, and one unique page per learned spell.
- Spell imagery is generated from numeric spell values from The Spell Writing Guide fields (no image URL required).
- Spellbook and wizard profile are persisted in browser localStorage.

## Run
Open `index.html` in a browser.
