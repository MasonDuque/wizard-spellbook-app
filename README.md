# wizard-spellbook-app

A lightweight browser-based D&D wizard spellbook.

## Features
- Wizard cover page with name, class, and level.
- Prepared spell slot tracking based on wizard level.
- Blank spellbook start with table of contents.
- Dedicated "Learn a Spell" page with structured spell input.
- Numeric capture fields inspired by **The Spell Writing Guide** by **The Gorilla of Destiny**.
- A generated page for every learned spell, including spell imagery.
- Local persistence in browser `localStorage`.

## Run
Open `index.html` in a browser.
